# 📸 Instagram Login Page – Clone

Projeto que replica a interface da página de login do Instagram, com foco em **layout fiel**, **responsividade** e **simulação de interação**. Desenvolvido com **HTML5** e **CSS3** puro (sem frameworks).

## 🎯 Funcionalidades

- Interface idêntica à página inicial de login do Instagram
- Campos para **usuário/e-mail** e **senha**
- Botão **"Entrar"**
- Opção **"Entrar com o Facebook"**
- Link para **"Esqueceu a senha?"** e **"Cadastre-se"**
- Seção **"Obtenha o aplicativo"** com badges da App Store e Google Play
- **Design responsivo** – adapta-se a desktop, tablets e smartphones
- Esconde o mockup do celular em telas pequenas para melhor usabilidade

## 🛠️ Tecnologias utilizadas

- HTML5 semântico
- CSS3 (Flexbox, Grid, Media Queries, Responsividade)
